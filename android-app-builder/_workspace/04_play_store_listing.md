# 04 Google Play 등록 초안

## 앱 이름 후보
- 레인린 (RainLean)
- 우산가이드

## 한줄 설명
- 비 오는 날, 내 위치와 바람을 분석해 우산 기울임 방향을 안내합니다.

## 상세 설명(요약)
- 현재 위치의 강수량과 바람 방향을 바탕으로 우산을 어느 방향으로 기울여야 비를 덜 맞는지 안내합니다.
- 3D처럼 기울어지는 우산 시각 요소로 직관적인 방향 피드백을 제공합니다.
- 서버 없이 단말에서 계산하며, 필요한 경우에만 날씨 API를 호출합니다.

## 권한 사유
- 위치 권한: 현재 위치 날씨 조회
- 센서 권한(자이로/회전벡터): 사용자 진행 방향 기준 안내
- 인터넷: 날씨 API 호출

## 정책 체크
- 사용자 위치 데이터의 보관 정책 명시
- 광고 SDK 미탑재 시 광고 관련 고지 제거
- 개인정보처리방침 URL 준비
